# WanderDoo

A clean and responsive travel website built with HTML, CSS, and JavaScript.

## Overview

This is a multi-page travel website featuring a minimalist design inspired by Apple's aesthetic. The website includes three main pages: Home, About, and Contact, with full mobile responsiveness.

## File Structure
```
travel-website/
├── index.html
├── about.html
├── contact.html

```

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Mobile hamburger menu for small screens
- Contact form with validation
- Smooth animations and transitions
- Clean, minimalist interface with limited color palette
=
## Pages

**Home Page**
- Hero section with call-to-action
- Featured destinations section with three travel locations

**About Page**
- Company mission and vision
- Three content sections explaining the website's purpose

**Contact Page**
- Contact form with name, email, gender, country selection, and message fields
- Contact information and social media links

## Setup

1. Clone this repository
2. Ensure all files are in the correct structure as shown above
4. Open `index.html` in a web browser

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome 6.4.0 (for icons)

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, Edge, and Opera.



## Contact Form

The contact form currently shows an alert message on submission. To connect it to a backend service, you will need to add server-side processing or use a form service like Formspree or Netlify Forms.
